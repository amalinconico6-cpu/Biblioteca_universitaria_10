void it.unisa.bibliotecajfx.controller.ControllerListaPrestiti.onModificaPrestito ( ActionEvent event) throws IOException
private

Gestisce la modifica di un prestito selezionato.

Apre una finestra modale di modifica (ModificaPrestito.fxml), passa al controller di modifica le liste e il prestito selezionato. Dopo la chiusura della finestra, aggiorna la tabella per mostrare eventuali cambiamenti.

Parameters
event
Exceptions
IOExceptionErrore nel caricamento dell'FXML
Precondition
Deve essere selezionato un prestito nella tabella
Postcondition
La tabella viene aggiornata

◆ onRimuoviPrestito()

void it.unisa.bibliotecajfx.controller.ControllerListaPrestiti.onRimuoviPrestito ( ActionEvent event) throws IOException
private

Gestisce la rimozione del prestito selezionato.

Procedura: 1) Controlla che un prestito sia selezionato 2) Aggiorna la copia sul libro associato (+1 copie) 3) Rimuove il prestito dalla listaPrestiti 4) Salva prestiti e libri su file 5) Aggiorna la tabella

Parameters
event
Exceptions
IOExceptionErrore durante il salvataggio su file
Precondition
Deve essere selezionato un prestito nella tabella
Postcondition
Se la rimozione va a buon fine il prestito non esiste più e le copie sono aggiornate

Imposta i riferimenti ai modelli necessari alla schermata.

Assegna al controller le liste di prestiti, utenti e libri. Riempie la TableView con i prestiti presenti e forza un refresh.

Parameters
lpLista_Prestiti da visualizzare/gestire
luLista_Utenti usata per controlli o modifiche
llListaLibri usata per aggiornare le copie dei libri
Precondition
lp != null && lu != null && ll != null
Postcondition
La tabella contiene i prestiti presenti in lp
